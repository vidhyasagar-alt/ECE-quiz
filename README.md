## ✨ What is this project?

A fully functional, browser-based **MCQ Quiz Platform** designed exclusively for ECE students. It features a slick circuit-board themed UI, real-time timer, live question tracking, Google Sheets integration for result collection, and a password-protected admin dashboard — all with **zero backend, zero frameworks**.

> 🖥️ **No server needed — runs entirely in the browser!**

---

## 🚀 Live Features

| Feature | Details |
|---------|---------|
| 📝 **Question Bank** | 200 MCQs across 4 randomized sets (Anna University Reg. 2021) |
| ⏱️ **Countdown Timer** | 25-minute live timer with color-coded warnings |
| 📊 **Question Tracker** | Visual panel showing answered / unanswered / skipped |
| 🔀 **Randomized Sets** | One of 4 question sets randomly assigned per attempt |
| 📈 **Google Sheets** | Results auto-saved via Google Apps Script |
| 🔐 **Admin Panel** | Password-protected dashboard with rankings & CSV export |
| 🏆 **Leaderboard** | Ranked results with score bars and medals |
| 📱 **Responsive** | Works on desktop, tablet, and mobile |

---

## 🕹️ How It Works

```
📋 Student fills Registration Form (Name, College, Year, ECE Dept, Email, Phone)
        ↓
✅ Validation — Only ECE Department students can proceed
        ↓
📖 Welcome Screen — Quiz rules and info displayed
        ↓
⚡ Quiz Starts — 50 random MCQs, 25-minute timer begins
        ↓
🖐️ Student answers questions, navigates via sidebar panel
        ↓
✔️ Submit — Confirmation modal shows answered / skipped stats
        ↓
📊 Results sent to Google Sheets + saved to localStorage
        ↓
🎓 Thank You screen displayed to student
```

## ⚙️ Installation & Setup

### ✅ Prerequisites
- Any modern web browser (Chrome, Firefox, Edge)
- No installations needed!

### 📥 Step 1 — Clone the Repository
```bash
git clone https://github.com/vidhyasagar-alt/ece-technical-quiz.git
cd ece-technical-quiz
```

### ▶️ Step 2 — Run the Project
```bash
# Simply open the file in your browser
open index.html
```
Or just **double-click** `index.html` — that's it! 🎉

### 🔐 Step 3 — Access Admin Panel
- On the registration page, type **`admin`** anywhere on your keyboard (secret shortcut)
- Enter the admin password to view the results dashboard
